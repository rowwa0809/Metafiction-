# Emergence Report

Generated by running `node test/run-sim.mjs` — a headless, DOM-free
harness that runs the exact same simulation modules the browser uses
(no mocking, no scripted outcomes). Every result below is the actual
output of that run, not a hand-authored example.

## Test 5 — Determinism (same seed -> identical history)

Two independently-created worlds with seed `42`, run for 10 in-game days
with no player input:

```
timelines identical: true
npc states identical: true
```

**PASS.** Every stochastic decision in the simulation routes through the
single seeded `world.rng` stream (or a named `.child()` sub-stream at
worldgen), so two runs of the same seed produce byte-identical event
timelines and NPC state (position, needs, everything).

## Test 1 — 30 in-game days, no player input

Seed `20260714`, run for 30 days with zero player interaction.

- **New friendships:** 51 (relationship-milestone detection crossing
  sentiment > 0.35 with familiarity > 0.15). Sample: *"Aldric Cooper and
  Orla Cooper have become friends."*, *"Konrad Shepherd and Jorah Shepherd
  have become friends."* Most emergent friendships in this 30-day window
  were among family/coworkers, since they interact far more frequently
  than strangers do — a realistic consequence of the interaction-frequency
  model, not a scripted outcome. Longer runs would be expected to surface
  friendships between unrelated villagers too, as their sentiment/
  familiarity slowly accumulates.
- **Conflicts:** 2 rivalries emerged from grudges, e.g. *"Doran Fletcher
  now considers Garrick Fletcher a rival."* / *"Garrick Fletcher now
  considers Doran Fletcher a rival."* — a mutual falling-out between two
  people who started out on good terms, driven entirely by ambient
  conversation outcomes (an accusation or insult that occurred organically
  during routine socializing).
- **Rumor spread:** the most-propagated belief in the population reached
  6 separate NPCs who hadn't witnessed the originating event directly
  (`npc_34|isATheif|true` — spread via gossip/overhearing after a
  theft was witnessed).
- **Price changes driven by supply:** 4, including a shortage-driven bread
  price rising from 4.28 -> 5.97 -> 9.88 coin as the grain->bread chain
  came under strain, and a tools glut pushing that price down from 10 to
  6.98.
- **Festivals:** 2 (The Planting Festival, Midsummer Fair) fired on their
  calendar dates and produced a shared high-importance memory for every
  attending NPC.
- **Thefts:** 12 — the shortage -> hunger -> desperation -> theft link in
  the cascade requirement, tuned down from an earlier, overly punishing
  economy pass (see "Balance notes" below).

**PASS** (>=1 friendship, >=1 price change, rumor reaching 5+ NPCs all
satisfied; conflict and theft cascades also observed within the same 30
days).

## Test 2 — Two witnesses, same event, diverge after 10 days

Two NPCs were given matching memory records of a fabricated shared event
(confidence forced to 0.5 to make them retrieval-distortion-eligible),
then had that memory retrieved repeatedly over 10 simulated days
(modeling occasional recall/retelling):

```
before: {"a":["npc_2"],"av":0.3,"b":["npc_1"],"bv":0.3}
after:  {"a":["npc_2"],"av":0.33056953003630046,"b":["npc_1"],"bv":0.4827349760177273}
```

**PASS.** Witness B's remembered emotional intensity of the event drifted
much further (0.30 -> 0.48) than witness A's (0.30 -> 0.33) purely from
independent reconstructive-distortion rolls on each retrieval — the same
mechanism (`memory.js#distort`) that can also swap a misremembered
participant. Two people who saw the same thing no longer remember it the
same way.

## Test 3 — Insult, leave for 5 days, return

The player insulted an NPC while a second NPC stood within earshot, then
5 in-game days passed with no further player input:

```
sentiment before: 0.00  after: -0.97
target has grudge: true
a nearby NPC witnessed/overheard it: true
```

**PASS.** The insulted NPC's relationship toward the player collapsed
from neutral to strongly negative and a grudge record was created — since
greeting-intent selection buckets by relationship sentiment (`< -0.25` ->
the "cold" greeting bank in `dialogue.js`), this NPC's baseline greeting
is now demonstrably different from before the insult. The bystander NPC
independently picked up the incident through `perception.js`'s
overhearing channel, recording both an episodic memory and a
`treatedPoorly` belief about the player — and that belief, being a
belief rather than a low-importance episodic memory, survives nightly
consolidation even if the raw memory of overhearing it eventually decays.

## Test 4 — Save/load + catch-up simulation

```
round-trip clock matches: true
npc count matches: true
catch-up advanced clock: true
```

**PASS.** `serializeWorld` -> `JSON.stringify` -> `JSON.parse` ->
`deserializeWorld` round-trips the clock and full NPC population exactly,
and `catchUpSimulation` (the same `tickWorld` code path used for normal
play, just run in a tight loop) advances the restored world by the
requested elapsed time. In the browser, `main.js` offers this as a
"simulate elapsed time" prompt when the saved timestamp is more than 30
real seconds old.

## Test 6 — Simulation-awareness reactions

Three NPCs, picked to span low/mid/high (openness + intelligence), were
each told "you are living inside a simulation, a video game":

```
Aldric Carter  — openness 0.27, intelligence 0.47 -> curiosity          (belief confidence after: 0.93)
Lucan Miller   — openness 0.52, intelligence 0.46 -> confusion          (belief confidence after: 0.92)
Nessa Fisher   — openness 0.82, intelligence 0.66 -> existentialCrisis  (belief confidence after: 0.91)
```

**PASS.** All three reacted differently, all three reactions are
explainable directly from inspector-visible state (their openness/
intelligence/neuroticism and the resulting confidence drop in their
`world.isReal` belief — see `beliefs.js#evaluateSimulationClaim`), and
none instantly accepted (`world.isReal` confidence stayed above 0.9 for
all three after a single, one-off claim — acceptance requires it to drop
below 0.5, which takes repeated strong evidence).

## Test 7 — Perception-limited world awareness

An NPC on one side of the village directly witnessed a scripted "fire"
event; an NPC placed at the opposite corner of the map was checked
immediately afterward:

```
far NPC knows immediately (should be false): false
```

**PASS.** The far NPC's memory table has zero record of the event. The
only way it can find out is the same way anything spreads in this
simulation: a witness has to physically carry it into a conversation
within gossip/overhearing range (see `perception.js` and
`dialogue.js#recordOverhearing`).

## Balance notes (tuning found during verification)

A few numbers were adjusted after the first full run surfaced real
imbalances (not narrated here as "planned," they were bugs/mistuning
caught by actually running the sim):

- **Building-lot bug:** the map generator originally had 12 small
  buildings but only 8 small lots, silently doubling up two buildings on
  one tile (caught visually in a browser screenshot, not by any
  headless test). Fixed by splitting lots into `large`/`workshop`/`home`
  categories sized to match `BUILDING_DEFS` exactly.
- **Zero friendships in an early run:** family relationship edges started
  with high sentiment but familiarity crept up too slowly (+0.03/
  interaction, and conversations were rarer than intended) to ever cross
  the milestone threshold in 30 days. Fixed by widening the conversation
  radius (2 -> 3 tiles), letting coworkers/neighbors chat during work and
  errands (not only dedicated "social" time), adding a small chance of
  a housemate chat during the sleep block, raising the familiarity gain
  per interaction, and slightly loosening the friendship threshold.
- **Theft rate spiked to 30-35/month** after households were made to
  actually pay coin for bread (a fix in itself — they previously "bought"
  bread without spending any money, which silently defeated the
  shortage -> desperation -> theft cascade). Retuned the theft
  probability and starting coin range down to a level (12/month) that
  still clearly demonstrates the cascade without turning the whole
  village into thieves.


numbers for test 1 (seed `20260714`) and test 5 (seed `42`).
